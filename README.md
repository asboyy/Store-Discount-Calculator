# Store Discount Calculator

Store Discount Calculator adalah aplikasi desktop yang dirancang untuk membantu pemilik toko atau kasir dalam menghitung nilai diskon dan harga akhir produk secara cepat. Aplikasi ini sangat berguna untuk memproses promosi penjualan tanpa perlu menghitung secara manual.

## 🚀 Fitur Utama

- **Perhitungan Diskon Otomatis:** Menghitung besar potongan harga berdasarkan persentase yang dimasukkan.
- **Harga Akhir (Net Price):** Menampilkan total harga yang harus dibayar setelah dikurangi diskon.
- **Input Fleksibel:** Mendukung pengisian harga barang dan persentase diskon sesuai kebutuhan promosi.
- **Antarmuka Sederhana:** Desain yang fokus pada kecepatan input dan kejelasan hasil perhitungan.

## 🛠️ Teknologi yang Digunakan

- **Bahasa Pemrograman:** Visual Basic .NET (VB.NET)
- **Framework:** .NET Framework
- **IDE:** Visual Studio

## 📋 Rumus yang Digunakan

Aplikasi ini bekerja dengan logika matematika sederhana namun akurat:
1. **Nilai Diskon** = Harga Asli $\times$ (Persentase Diskon / 100)
2. **Harga Akhir** = Harga Asli $-$ Nilai Diskon
